# Site Psicólogo Danilo Souza

Landing page institucional do Psicólogo Danilo Souza, com foco em apresentação profissional, áreas de atendimento e canais de contato.

## Tecnologias

- HTML5
- CSS3
- JavaScript (vanilla)

## Funcionalidades

- Layout responsivo para desktop e mobile
- Navegação por seções (`Sobre`, `Abordagem`, `Temas`, `Benefícios`, `Contato`)
- Botões de contato com WhatsApp e Instagram
- Alternância de tema claro/escuro
- Tema escuro com base preta (`#000`) e acentos em verde esmeralda
- Persistência de preferência de tema no `localStorage`

## Estrutura do projeto

```text
danilo-souza/
  assets/
  index.html
  styles.css
  script.js
```

## Como executar localmente

Como é um projeto estático, basta abrir o `index.html` no navegador.

Opcionalmente, você pode usar uma extensão de servidor local (ex.: Live Server no VS Code) para facilitar o desenvolvimento.
